# Windows


## WIN +:
```
R       run
E       explorer
up      maximize
down    minimize
D       desktop
M       minimize all
Tab     task view
```

## ALT +:
```
Tab     switch programs
Esc     switch programs
letter  menu items
F4      close
space   menu	
```

## CTRL +:
```
Esc                     start
left/right/up/down      move cursor
CTRL + SHIFT + ESC      task manager
```	

## File explorer
```
End         bottom
Home        top
left        collapse
right       current
```

## Others
```
F2          rename
F5          refresh
```
